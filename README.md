Jarvis Voice Assistant

Overview

Jarvis Voice Assistant is a Python-based interactive chatbot that supports both text and voice communication. The system accepts user input through the keyboard or microphone, processes the request, and responds using both text and speech.

Features

- Dual Input Support (Text and Voice)
- Speech-to-Text Conversion
- Text-to-Speech Response
- Console-Based Interaction
- Conversation Logging with Timestamps
- Date and Time Information
- Website Opening Commands
- Simple and User-Friendly Interface

Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- PyAudio

How to Run

1. Install the required libraries:
   pip install SpeechRecognition pyttsx3 pyaudio
2. Run the program:
   python chatbot.py

Supported Commands

- hello
- your name
- time
- date
- open google
- open youtube
- how are you
- exit

Conversation Logging

All conversations are automatically stored in "conversation_log.txt" with timestamps for future reference.
